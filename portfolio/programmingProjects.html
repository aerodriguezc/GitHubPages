MultiActions

Features:

  • Find elements closer to others and relate them by exchanging parameters values.
  • Works in elements in the model and between local elements and linked models.
  • Establish geometrical relationships between elements: Equipment pads above equipments, electrical equipments at the start and end of cable trays, mechanical equipments at the start and end of pipes.
  • It is a multimodel application which means you can select multiple models and this action will be performed in each of them automatically.

Revit Data Reader

  • Centralize the information of all your models and display it in another application (descktop or web based).

Revit Drawings Automation

Features:

  • Automate the creation of drawings in Revit based on engineering information.
  • This application creates certain types of drawings based on tables in a database (engineering data).
  • Drawings based on detail items so each element or symbol has information (properties) in it.
  • Drawings become a source of information, not only lines but countable and comparable elements with information.
  • Accelerate the process of drawings creation and remove inconsistencies between drawings and engineering lists.

Tekla Control Web App

Features:

  • Web application that extract information from a Tekla model and store it in a database.
  • Group the plates information by thickness.
  • Group elements by type of profile and calculate the number of bars needed for fabrication.
  • Calculate all the combination of plate sizes to buy in order to fabricate the structure.
  • Allow to control the production of elements by changing the status of teh assemblies.
  • Is web based so all the information can be accessed from anywhere in the world and shared in real time.

Spatial Analysis

Features:

  • This application clusters data from a Tekla model based on his position in the model.
  • Clustering is a machine learning algorithm that discovers patterns and groups without human intervention.
  • This can be used to group elements by fabrication sets and then by mountage sets.
  • Sets are defined by the algorithm and then refined by the user.
  • Sets can be defined by any property of the elements not only by their spatial position.
